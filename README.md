## Sobre o Projeto
Beautiful-Design é uma construção visual criada com Python usando a biblioteca Turtle, explorando formas geométricas, cores em gradiente e movimentos precisos.  
O projeto tem como foco demonstrar domínio de lógica, estética visual e organização profissional no desenvolvimento.

## Objetivo
Gerar um padrão artístico dinâmico a partir de cálculos de movimento rotacional e variação progressiva de cores, produzindo uma composição visual única e simétrica.

## Tecnologias Utilizadas
- **Python 3**
- **Turtle Graphics**

## Estrutura
O código principal responsável pela geração visual está armazenado no arquivo dedicado dentro do repositório, mantendo organização clara para análise técnica.
